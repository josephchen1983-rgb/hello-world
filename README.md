# MQTT-Web 项目

一个完整的物联网数据可视化平台，集成 MQTT、Python、MySQL、InfluxDB、WebSocket 和 ECharts，实现实时数据展示和管理。

## 🏗️ 项目架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Web 浏览器 (ECharts)                      │
│              Admin 管理端 | User 用户端                      │
└────────────┬────────────────────────────────────────────────┘
             │ WebSocket (实时数据)
┌────────────┴────────────────────────────────────────────────┐
│         Python 后端 (Flask/FastAPI)                         │
│  ├─ 认证系统 (JWT)                                          │
│  ├─ WebSocket 服务                                          │
│  ├─ MQTT 客户端                                             │
│  └─ 数据处理引擎                                            │
└────────────┬────────────────────────────────────────────────┘
             │
    ┌────────┴────────┬─────────────┐
    │                 │             │
┌───▼────┐      ┌────▼────┐    ┌───▼─────┐
│  EMQX  │      │  MySQL  │    │InfluxDB │
│ (MQTT) │      │ (用户等) │    │(时序数据)│
└────────┘      └─────────┘    └─────────┘
```

## 📋 技术栈

- **MQTT Broker**: EMQX
- **后端**: Python (Flask/FastAPI)
- **数据库**: MySQL (用户认证、配置) + InfluxDB (时序数据)
- **实时通信**: WebSocket
- **前端**: HTML5 + JavaScript + ECharts
- **容器化**: Docker + Docker Compose

## ✨ 主要功能

### 管理员 (Admin)
- 👥 用户管理
- 🔧 设备配置管理
- 📊 完整的数据可视化
- 📈 数据分析和报表
- ⚙️ 系统设置

### 普通用户 (User)
- 📊 查看实时数据
- 📈 查看个人设备数据
- 💾 数据导出
